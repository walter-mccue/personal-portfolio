WEB-231 Enterprise JavaScript I

Visit my WEB-231 GitHub Repository

Visit my WEB-231 GitHub Landing Page

This is the introductory course into JavaScript. I took the basics I had learned in HTML and CSS and started building web pages with functional JavaScript within them. Every assignment is structured and styled similarly but the difference is the content and scripting. Assignments started as simple input/selection forms with submit buttons to more complicated random card deck shuffling.

WEB-330 Enterprise JavaScript II

Visit my WEB-330 GitHub Repository

Visit my WEB-330 GitHub Landing Page

As the name suggests, this is a more advanced JavaScript Course. More complex assignments with very structured styles and even a light/dark theme option. These assignments required much more research and understanding of various JavaScript syntax. It also required learning how to troubleshoot code to find any code with errors or extra/missing code.

Pets-R-Us

Visit my Pets-R-Us GitHub Repository

Visit my Pets-R-Us Deployment Page

This project focused on using Node.js and Express and a few other dependencies (installed via npm) to build and launch a website. We also had to create database collections and control the flow of data between the database and the local web server. The website will be live for as long as Heroku can host the project.
